# Вёрстка проекта

## Структура

В папке `dist/` исходники `html, css, js`

В папке `src/` исходный код `pug, sass, js`

## Для разработки

- `npm ci` or `npm i`
- настройте путь к файлам `--public-url *you_url*` в `package.json` строка `build`
- собрать проект `npm run build`
- дев режим проекта `npm start`
